Poncho

La librería de estilos y componentes oficiales para diseñar y desarrollar sitios web y aplicaciones móviles del Gobierno de la República Argentina.

¿Por qué usar Poncho?

Experiencia unificada

Los ciudadanos van a reconocer el sitio como parte de los sitios oficiales del Gobierno.

Sitios accesibles

Los componentes fueron mejorados para que los usen personas con discapacidad.

Menos tiempo de trabajo

Copiá y pegá componentes y plantillas sin necesidad de desarrollar desde cero.

¿Cómo instalar Poncho?

Poncho usa de base los estilos y scripts de Bootstrap 3.

También tenemos una prueba de Poncho para Bootstrap 4. Estamos trabajando en la documentación completa.

Si usás NPM

Ejecutá en la consola el comando npm i ar-poncho

Si descargás los archivos manualmente

Creá archivos html con estas dependencias:

CSS - Bootstrap (v-3.4.1), Font Awesome (v-4.7.0), Poncho, Iconos

<link href="https://stackpath.bootstrapcdn.com/bootstrap/3.4.1/css/bootstrap.min.css"  rel="stylesheet">
<link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css" rel="stylesheet">
<link href="dist/css/poncho.min.css" rel="stylesheet">
<link href="dist/css/icono-arg.css" rel="stylesheet">

JavaScript - Bootstrap (v-3.4.1), JQuery

<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/2.2.4/jquery.min.js></script>
<script src="https://stackpath.bootstrapcdn.com/bootstrap/3.4.1/js/bootstrap.min.js”></script>

Todas las dependencias son rutas absolutas excepto poncho.min.css e icono-arg.css, que los debés descargar del repositorio de Poncho en github y usar de manera local.
